1. 这一版为入门前端时练习用。
2. 代码中的命名格式
 + 代码中所有含多个单词的id以“name_name”的格式命名。
 + 所有含多个单词的class以“name-name”的格式命名。
 + 所有含多个单词的文件夹（folder）以“name-name”的格式命名，（搜索引擎对“-”的敏感度较高）。
 + 函数（function）以驼峰格式（nameNameName）的格式命名。
